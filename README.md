# 🍔 Swiggy Sales Analysis Dashboard (Excel)

## 📊 Overview
This project focuses on analyzing Swiggy sales data using Microsoft Excel to uncover meaningful business insights related to customer behavior, sales trends, and product performance.

The goal of this project is to transform raw data into actionable insights through data cleaning, analysis, and interactive dashboard creation.

---

## 🚀 Key Features

- Data cleaning and preprocessing in Excel  
- Interactive dashboard using Pivot Tables & Charts  
- Sales performance analysis  
- Customer behavior insights  
- Order trend visualization  

---

## 🛠️ Tools & Technologies

- Microsoft Excel  
- Pivot Tables  
- Charts & Data Visualization  
- Data Cleaning Techniques
